# חדיש - משחק המתנות 🎁

משחק דפדפן לאיסוף מתנות, מבוסס על חנות חדיש.

## הפעלה ב-GitHub Pages

1. צור repository חדש ב-GitHub
2. העלה את הקובץ `index.html`
3. עבור להגדרות → Pages → בחר branch `main` → שמור
4. המשחק יהיה זמין בכתובת: `https://[שם-משתמש].github.io/[שם-repository]`

## תכונות

- 🎮 משחק תפיסת מתנות עם עגלה
- 💣 פצצות להימנעות
- ⭐ מתנות בונוס x2
- 🔥 מערכת קומבו
- 📈 רמות קושי עולות
- 🎟️ קופון לזוכים
- 🛠️ דף ניהול מובנה (לחצן ⚙️)

## דף ניהול

לחץ על כפתור ⚙️ בפינה שמאלית עליונה:
- עדכון לוגו

- הוספת מוצרים עם תמונות
